Our Technology

State-of-the-art deep learning for AI detection

Model Architecture

Convolutional Neural Network (CNN)

  • • Input Layer: 32×32 RGB images
  • • Hidden Layers: Multiple Conv2D layers with BatchNormalization
  • • Pooling: GlobalAveragePooling for dimensionality reduction
  • • Output: Binary classification (Real/Fake)
  • • Framework: TensorFlow/Keras 3.14.1

Training Dataset

CIFAKE Dataset (60,000 samples)

Real Images: 30,000 (50%)
AI-Generated: 30,000 (50%)

Performance Metrics

Overall Accuracy: 92.98%
Inference Latency: 0.8s
Model Size: 918 KB

Explainability: Grad-CAM

Gradient-weighted Class Activation Maps (Grad-CAM) provide visual explanations of model predictions by highlighting the regions that most influenced the decision.

  • • Generates attention maps showing important pixels
  • • Reveals which artifacts the model detected
  • • Increases transparency and trust in predictions
  • • Helps identify AI generation patterns
